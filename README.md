# 🍬 Glycemic Analytics App

A Streamlit-based machine learning web application for analyzing glycemic data and making predictions.

---

## 🚀 Features
- Data analysis and visualization
- Machine learning predictions
- Interactive dashboard using Streamlit
- Modular and clean code structure

---

## 🛠️ Technologies Used
- Python
- Streamlit
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

---

## 📁 Project Structure
- app.py → Main application
- modules/ → Backend logic (ML, processing, visualization)
- assets/ → Images and UI files
- data/ → Dataset files

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
